The winning entries


Vilsol

1

Out of all the submissions we received, Vilsol's was our favourite and as such the entry has been awarded first place. Vilsol's plugin allows for a number of mechanics (that you'd normally see in a client mod) to be applied to entities. From conveyor belts and cannons to teleporters, Vilsol's plugin adds a lot of depth and flexibility for those creating redstone systems. The repository README includes fantastic documentation on what is possible with the plugin and a series of GIFs that demonstrate the available functionality. We liked the creativity and quality of the plugin, and overall it was a great submission.

slipcor (BowsPlus)

2

slipcor was awarded second place for his submission on the theme "entities". What started off as a simple idea: firing creepers from bows, quickly grew into what is now BowsPlus. The plugin is fully configurable and allows you to customise (up to the finest details) the entities which are fired. From firing primed TNT (explosive arrows) to charged creepers (could be used as a weapon on a PvP server) and much more, the plugin can be used in a variety of different situations or just for fun. It supports firing falling blocks and can be configured to allow or disallow a list of entities either via the config or with permissions. Overall, a simple idea but effective idea.

mncat77 (FictionWorld)

3

mncat77 was awarded third place for their submission on the theme "books". The plugin allowed the player to generate a new world based entirely on a book. The generator looked at the book and evaluated its contents to create a whole new type of terrain that the user would never have encountered before. This allowed for a user to easily generate a new and wonderful world with unique contents easily and have great fun exploring it! We were impressed with the originality of the concept that the plugin was based around, as well as the fun we had testing the plugin and the general quality of the code produced. Overall, it was simply a fantastic submission.
